# org/example

Основной package upload-сервиса.

## Классы

- `Main.java` - стандартный `@SpringBootApplication`.
- `FileUploaderApplication.java` - контроллер с endpoint `POST /api/upload`.

## Рядом лежат нестандартные ресурсы

- `resources/application/application.properties`
- `resources/static/index.html`

Запуск идет через `gradlew.bat bootRun` из `B:\Local_AI_Projeckt\webServer\webServer`.
