# src/main

Основной код и ресурсы Spring Boot upload-сервиса.

## Что лежит внутри

- `java/org/example/` - Java-классы приложения.

Обратите внимание: ресурсы в текущем проекте лежат внутри `src/main/java/.../resources`, а не в стандартном `src/main/resources`. Это нестандартная структура, но именно так устроен текущий репозиторий.
