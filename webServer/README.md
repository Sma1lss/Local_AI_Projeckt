# webServer

Внешняя обертка для минимального Spring Boot upload-сервиса.

## Что здесь важно

Рабочий Gradle-проект лежит не в корне этой папки, а в `webServer/webServer/`.

## Назначение

Папка нужна как контейнер под простой Java backend, который сохраняет загруженный файл на диск. Это не полноценный downstream для FSSS, а минимальный учебный пример.
