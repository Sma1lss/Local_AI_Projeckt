# security_server_and_router_to_ai

Внешняя папка для Java-сервиса безопасной загрузки файлов.

## Что здесь важно

Рабочий Gradle/Spring Boot проект находится в `security_server_and_router_to_ai/untitled/`.

## Что еще лежит рядом

- `.idea/` - IDE-конфиги.
- `untitled/` - реальный проект.
- вложенные распакованные копии и архивы - не считать частью production-структуры.

Для запуска и полной структуры использовать README внутри `untitled/`.
