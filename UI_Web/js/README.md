# js

Каталог для клиентской логики фронтенда.

## Файлы

- `api.js` - предполагаемый слой запросов к backend.
- `chat.js` - логика страницы чата.
- `disciplines.js` - логика выбора дисциплин или тематик.
- `profile.js` - логика профиля.

## Состояние

Все файлы сейчас пустые. Папка задает будущую структуру frontend-кода, но не содержит рабочей реализации.
